================================================================================
TECHNICAL ENHANCEMENTS
================================================================================

1. Performance Optimizations

   - Image optimization (use responsive images, WebP format)
   - CSS minification (add 11ty plugin)
   - JavaScript minification (if you add JS)
   - Critical CSS inlining
   - Add service worker for offline support

2. Add Image Processing

   Install 11ty Image plugin:
   npm install --save-dev @11ty/eleventy-img

   Then configure in .eleventy.js for automatic image optimization

3. Add RSS Feed

   For blog or news updates:
   - Create feed.njk template
   - Configure in .eleventy.js
   - Link in HTML <head>

4. Improve Accessibility

   - Add proper ARIA labels
   - Ensure keyboard navigation works
   - Test with screen readers
   - Add skip links
   - Improve color contrast if needed

5. Add Security Headers

   Create _headers file for enhanced security:
   - Content-Security-Policy
   - X-Frame-Options
   - X-Content-Type-Options
   - Referrer-Policy

   Note: GitHub Pages has limited support for custom headers
   Consider Netlify or Cloudflare Pages for full control

QUICK REFERENCE - COMMON TASKS
================================================================================

Update Homepage:
  $ nano src/index.md
  $ git add src/index.md
  $ git commit -m "Update homepage content"
  $ git push origin main

Update Styles:
  $ nano src/css/style.css
  $ git add src/css/style.css
  $ git commit -m "Update styles"
  $ git push origin main

Add New Page:
  $ nano src/newpage.md
  Add frontmatter: layout, title, description
  $ git add src/newpage.md
  $ git commit -m "Add new page"
  $ git push origin main

Add Client Logo:
  1. Add image to src/images/client-name.png
  2. Edit src/_data/clients.json
  3. Commit and push

Test Locally:
  $ npm start
  Visit http://localhost:8080

Build for Production:
  $ npm run build
  Output in _site/ directory
